# Бот «Воздух НСК» для мессенджера MAX

Консультант по вентиляции и кондиционированию. Отвечает на вопросы, даёт контакты.

## Быстрый старт

1. Получи токен бота в MAX (см. [docs/MAX-BOT-SETUP.md](../docs/MAX-BOT-SETUP.md))
2. Скопируй `.env.example` в `.env` и укажи `MAX_BOT_TOKEN`
3. Запусти: `npm start`

## Команды

- `/start` — приветствие
- «контакты» — телефон, email, адрес
- «вентиляция», «кондиционер», «сервис», «цена» — краткие ответы по теме

## Структура

```
max-bot/
├── index.js      # основной код бота
├── package.json
├── .env          # MAX_BOT_TOKEN (не в git)
└── .env.example
```
