# Аудит проекта — март 2025

## Выполненные исправления

### 1. Прокрутка и производительность
- **Троттлинг scroll** (Navbar, BackToTop): добавлен throttle 100ms + RAF для ограничения re-render при скролле
- **Reveal-анимации**: уменьшен `amount` (0.08), `margin` (-40px), `duration` (0.45s), `y` (20px) — быстрее срабатывают и меньше нагружают
- **Backdrop-blur на мобильных**: уменьшена интенсивность (md: 12px→6px, xl: 24px→12px) для лучшей производительности GPU

### 2. Адаптивность и высоты блоков
- **Секции**: `py-10` на мобильных вместо `py-14`, плавный рост `md:py-24 lg:py-32`
- **Каталог**: карточки `h-[280px] sm:h-[320px] md:h-[400px] lg:h-[480px]`, `rounded-2xl` на мобильных
- **Наши работы**: карточки `w-[260px] sm:w-[308px]`, `h-[160px] sm:h-[198px]`
- **Hero**: `text-4xl sm:text-5xl`, `mb-6 md:mb-10`, `px-4 sm:px-6`
- **Container**: `px-4 sm:px-6` по всему сайту

### 3. «Наши работы» — затемнение при нажатии
- Состояние `pressedIndex`: при нажатии/касании карточки показывается затемнение (rgba 0.4)
- При начале перетаскивания `pressedIndex` сбрасывается
- `onPointerUp`/`onPointerCancel`/`onPointerLeave` очищают состояние
- Scale при hover уменьшен до 1.05 для более плавного ощущения

### 4. Touch-свайп на мобильных
- Обработчики `touchstart`, `touchmove`, `touchend`, `touchcancel` для горизонтальной прокрутки пальцем
- `touch-pan-x` для подсказки браузеру о горизонтальном жесте
